# DeepFake Image Detector

A Deep Learning-based web application that detects whether a face image is **Real** or **Fake** using **EfficientNetB0** and **Streamlit**.

## Features
- Upload image
- Detect Real / Fake
- Confidence score
- Built with TensorFlow + Streamlit

## Tech Stack
- Python
- TensorFlow
- EfficientNetB0
- Streamlit

## Run Locally
```bash
streamlit run app.py
